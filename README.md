# WebShopAAA

Är ett grupparbete som skapas av Anders, Ali och Ammar.

Det som vi skapar är en Webshop, som har ett backend i dotNET 6 MVC och ett frontend i React.

Medan fronend är i React så valde vi att skapa admin delen i MVC view för att slippa den kopplingen
till React samt för att inte behöva öppna upp det till API.
